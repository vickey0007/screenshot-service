
Environment Required:- Node.js v10.16.3

Steps to run:- npm i && node app.js

Steps to reproduce:-
1. Electron.js (JavaScript Framework for launching applications specific to user environment)
2. open -a "Google Chrome" (Command to open browser)
3. node-desktop-screenshot (Node.js module to take Screen Shots)
4. node-server-screenshot (Node.js module with everything above)
5. app.js (accept user input and save screenshots at default app location)
